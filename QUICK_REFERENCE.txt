BASIC LATEX FORMATTING
======================

BOLD:           \textbf{text}
ITALIC:         \textit{text}
MONOSPACE:      \texttt{text}
SMALL CAPS:     \textsc{text}

HEADERS:        \section{Title}
                \subsection{Title}
                \subsubsection{Title}


REFERENCES & CITATIONS
=====================

CITE A SOURCE:   \cite{keyname}
REFERENCE:       \ref{label}
EQUATION:        \eqref{eq:label}
FIGURE:          \ref{fig:label}
TABLE:           \ref{tab:label}

EXAMPLE:
  See Figure \ref{fig:myplot} and \cite{AuthorYear}


EQUATIONS
=========

INLINE:         $E = mc^2$

DISPLAY:        \begin{equation}
                E = mc^2
                \label{eq:einstein}
                \end{equation}

QUANTUM NOTATION:  $\braket{\psi|\phi}$
                   $|\psi\rangle$
                   $\langle\psi|$

FIGURES
=======

BASIC:          \begin{figure}[h]
                  \centering
                  \includegraphics[width=0.8\textwidth]{file.pdf}
                  \caption{Description}
                  \label{fig:label}
                \end{figure}

PLACEMENT OPTIONS:
  [h] - Here
  [t] - Top of page
  [b] - Bottom of page
  [p] - Separate page


TABLES
======

BASIC:          \begin{table}[h]
                  \centering
                  \begin{tabular}{|l|c|r|}
                    \hline
                    Left & Center & Right \\
                    \hline
                    Data & Data & Data \\
                    \hline
                  \end{tabular}
                  \caption{Title}
                  \label{tab:label}
                \end{table}

COLUMN OPTIONS:
  l - Left aligned
  c - Centered
  r - Right aligned
  | - Vertical line

COMMON ERRORS & FIXES
=====================

ERROR: "pdfinclude.sty not found"
FIX:   Already removed from template

ERROR: "Bibliography not showing"
FIX:   Did you run: pdflatex → bibtex → pdflatex → pdflatex?

ERROR: "Figure not displaying"
FIX:   Check image format (.pdf, .jpg, .png)
       Check filename spelling
       Is image in same directory as .tex?

ERROR: "Undefined control sequence \cite"
FIX:   Did you define the citation in Sample.bib?
       Check \cite{keyname} matches @article{keyname}

ERROR: "Page numbers wrong"
FIX:   Recompile twice
       LaTeX needs 2 passes to update page numbers

ADDING CITATIONS
================

1. Add entry to Sample.bib:
   @article{AuthorYear,
     author = {Author, A.},
     title = {Title},
     journal = {Journal},
     year = {2024}
   }

2. Cite in your text:
   \cite{AuthorYear}

3. Compile:
   pdflatex → bibtex → pdflatex → pdflatex

QUICK COMPILATION CHECKLIST
============================

Issue: PDF not updating after editing
→ Delete all auxiliary files (.aux, .toc, .out, .lof, .lot)
→ Recompile


Issue: Changes not appearing
→ In TeXstudio: Tools → Clean Auxiliary Files
→ Then F5 to recompile


Issue: "LaTeX Error"
→ Check red error in editor
→ Look at line number indicated
→ Search that line for syntax errors
→ Verify matching braces: { }
